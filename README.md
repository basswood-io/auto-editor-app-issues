# Auto-Editor Tracker
This is the open source tracker for Auto-Editor, a commerical 'open core' software product.

You can purchase the app here: https://auto-editor.com/app
Please use the email on the website for any financial or private concerns.

Use the Issues Page if you would like to report a bug.
Use the Discussions Page if you would like to chat with other users.

If your issue is about ae-core only. Please make an issue [on this repo](https://github.com/WyattBlue/auto-editor) instead.
